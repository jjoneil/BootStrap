FORMS!

Information inputs, buttons, and check/select boxes



Forms are a way for the user to input data on a web page. In order for that exchange to take place, the form would send data to a web server and do something with it. That's a little complicated, but building the forms with HTML/CSS is not. 






<label for="emailFormExample">Email Address</label>
<input type="email" class="form-control" id="emailFormExample" placeholder="Put ur email here look">

All are availible out of the box in html, but bootstrap styles the text and size. JS allows the users inputs to be used. Will be block level by default.


Other types include checkboxes, radio buttons, dollar amounts, text input boxes, selects

BUTTONS!


Buttons come in all shapes and sizes. Commonly used as links and inputs for forms. Inline by default. Bootstrap supports hover affects, shadows, and opacity. Buttons will appear pressed with the .active class.

INPUTS!


Inputs are an important way for the user to interact with the site, submitting and getting information in return. Inputs include text, password, datetime, datetime-local, date, month, time, week, number, email, url, search, tel, and color. The type="" specifies how the form will look.




	<input type="text" class="form-control" placeholder="Text input">
